#DressUp – Fashion for Everyone

DressUp is a modern e-commerce web application where users can explore and purchase clothing for all genders and styles. The platform is designed to provide a smooth shopping experience with a clean UI and responsive design.

#Features
----------------
👕 Browse clothing by category (Men, Women, Unisex)
🔍 Search and filter products easily
🛒 Add to cart and manage items
📱 Fully responsive (mobile + desktop)
⚡ Fast performance with Vite


Frontend: React.js
Build Tool: Vite
State Management: Context API / Hooks
